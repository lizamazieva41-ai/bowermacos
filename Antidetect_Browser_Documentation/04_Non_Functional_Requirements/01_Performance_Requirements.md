# 1. YÊU CẦU HIỆU NĂNG

| Thao tác | Thời gian tối đa |
|----------|-----------------|
| Khởi tạo profile mới | ≤ 5 giây |
| Điều hướng trang | ≤ 3 giây |

| Tài nguyên | Yêu cầu |
|------------|----------|

| CPU per profile | ≤ 10% |

| Chỉ số | Yêu cầu |
|---------|----------|
