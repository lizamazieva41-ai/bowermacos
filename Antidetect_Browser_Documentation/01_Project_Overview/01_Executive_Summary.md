# 1. TỔNG QUAN DỰ ÁN

## 1.1 Mục đích
Tài liệu này cung cấp tổng quan về dự án Antidetect Browser System V2.

## 1.2 Mục tiêu chính
- Tạo nhiều profile trình duyệt headless đồng thời
- Điều khiển từ xa qua API hoặc CLI
- Giả mạo fingerprint để tránh bị phát hiện
- Quản lý proxy riêng cho từng session

## 1.3 Lợi ích
- Tiết kiệm 50-70% RAM với headless mode
- Bảo mật cao với fingerprint/proxy riêng biệt
- Vượt qua anti-bot systems (Cloudflare, DataDome, reCAPTCHA)
