# 2. BẢNG THUẬT NGỮ

## Thuật ngữ chính

| Thuật ngữ | Định nghĩa |
|-----------|-------------|
| **Headless Mode** | Chế độ chạy trình duyệt không có GUI |
| **Fingerprint** | Dấu vết kỹ thuật số của trình duyệt |
| **CDP** | Chrome DevTools Protocol - Giao thức điều khiển Chrome |
| **Browser Context** | Phiên làm việc độc lập của trình duyệt |
| **Stealth Mode** | Chế độ ẩn danh để tránh bị phát hiện |
| **WebRTC** | Giao thức có thể rò rỉ IP thật |
| **Proxy** | Máy chủ trung gian để ẩn IP thực |
