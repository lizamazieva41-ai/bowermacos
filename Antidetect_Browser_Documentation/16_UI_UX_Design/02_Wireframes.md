### 6.1 Window Minimum Size

- **Width**: 1024px
- **Height**: 600px
- Below minimum: Display resize prompt

### 6.2 Sidebar Collapse

- **Expanded**: 240px (default)
- **Collapsed**: 64px (icons only)
- **Auto-collapse**: Below 1280px width
